Manul the Game is a Wii U application which shows you a picture of a Manul (Pallas' Cat.) <br>
You can post to Miiverse ([Roseverse](https://miiverse.projectrose.cafe) only) via the on-screen button. <br><b>Please note that there is no Roseverse community right now.<br><br>
<b>To install Manul the Game, extract your downloaded folder from releases, and then put the folder in your install folder on your Wii U's SD card, and install it using [NUSspli](https://github.com/V10lator/NUSspli) or any other prefered Wii U Title installer.</b>
