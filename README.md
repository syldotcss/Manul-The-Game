Manul the Game is a Wii U application which shows you a picture of a Manul (Pallas' Cat.) <br>
You can post to Miiverse (Roseverse only) via the on-screen button. <br><br>
<b>To install Manul the Game, put the folder in your install folder on your Wii U's SD card, and install it using NUSspli (or any other prefered Wii U Title installer.)</b>
